**1. Project Title**
SEVA SETU – Offline Healthcare Management System for ASHA Workers

**2. Problem Statement**
ASHA workers in rural areas often face poor or no internet connectivity, making real-time digital data collection difficult. Paper-based records are error-prone, inefficient, and prone to data loss.

**3. Solution**
SEVA SETU provides an offline-first digital healthcare system that allows ASHA workers to securely collect, store, and manage patient health data offline and automatically synchronize it when internet connectivity is restored.

**4. Key Features**
  - Offline patient data entry
  - Secure local data storage
  - Maternal & child health record management
  - Automatic data synchronization
  - Excel report export
  - Google Maps–based survey assistance
  - JWT-based authentication

**5. Tech Stack**
  - Frontend: React.js
  - Backend: Node.js
  - Offline Storage: IndexedDB / SQLite
  - APIs: REST (HTTPS)
  - Maps: Google Maps API
